# Kie AI SDK Plugin Creator

To build powerful sdk plugin for Kie-AI sdk, follow these steps:
1. Initialize the plugin directory using `references`
2. Develop the plugin by editing the copyed code

## When to Use This Skill

When you need to create a plugin for Kie AI SDK.

## Quick Start

### Step 1: Copy the Plugin Template

Copy the plugin template from the `references` to plugins folder.

```bash
cp -r references <plugin-name>
```

### Step 2: Develop the Plugin

Read the context by user input eg. documents, reference codes. and develop the plugin.
